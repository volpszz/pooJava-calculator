# ☕ Java Calculator

A simple calculator developed in Java using Object-Oriented Programming (OOP) concepts.

## 📋 Features
- Basic operations: addition, subtraction, multiplication and division
- Division by zero handling
- Repeat calculations without restarting the program

## 🗂️ Project Structure
```
oopJava-calculator/
├── src/
│   ├── calculus/
│   │   └── Calculus.java
│   └── Calculator.java
├── LICENSE
└── README.md
```

## 🚀 How to run
1. Clone the repository
```bash
git clone https://github.com/volpszz/oopJava-calculator
```
2. Open the project in IntelliJ IDEA
3. Run `Calculator.java`
